Task-Switching Game

Respond to the letters or digits based on the active rule. Use A (Left) for Vowels or Even digits, and L (Right) for Consonants or Odd digits.

How to Play & Examples

Each trial displays a digit-letter pair (e.g., G7). You must perform one of two tasks depending on the active rule:

  • Letter Task: Decide if the letter is a Vowel or Consonant.
  • Number Task: Decide if the digit is Even or Odd.

Key Mappings (Same for both tasks):

  • Left Key (A / ←): Vowel OR Even
  • Right Key (L / →): Consonant OR Odd

How to Know the Active Task:

  • Cued-Mixed Mode: A prompt (LETTER or NUMBER) appears above the stimulus.
  • Alternating Runs Mode: The quadrant position determines the task.
    • Top two quadrants = **Letter Task**
    • Bottom two quadrants = **Number Task**
  • Pure Blocks: Only one task is active for the whole block.

Example (with G7):

  • If active task is LETTER: Focus on G → Consonant → Press Right (L).
  • If active task is NUMBER: Focus on 7 → Odd → Press Right (L).
